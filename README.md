# 📚 Courses App

A React-based app to browse and filter courses with modern UI.

## ✨ Features
✅ Filter by category  
✅ Like courses (❤️)  
✅ Dark/Light mode  
✅ Retry button on API error  
✅ Spinner while loading  
✅ Share button (coming soon)  
✅ Responsive & Clean UI
